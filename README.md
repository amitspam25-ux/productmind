# ProductMind — AI PM Copilot

**Turn customer interviews, support tickets, and NPS feedback into PRDs and dev tickets in under 4 minutes.**

→ **[productmind.uk](https://www.productmind.uk)**

---

## What It Does

ProductMind is a local-first AI product management copilot. You drop in raw PM inputs — interview notes, NPS exports, support ticket CSVs, sales call transcripts — and it outputs:

- **Structured problem statements** with evidence citations
- **Full PRDs** (objectives, scope, requirements, acceptance criteria, success metrics)
- **Dev-ready ticket breakdowns** chained directly from approved requirements
- **Theme synthesis** across all customer signal sources

Every requirement links back to the source evidence. Every output improves the AI memory model for next time.

---

## Why Local-First?

ProductMind runs on [Ollama](https://ollama.com) models on your own machine.

- Zero data leaves your device
- No API keys, no cloud subscription for AI
- Safe for enterprise use — customer data, roadmap strategy, and competitive insight stay private

---

## Core Workflow

```
Customer signal (interviews / NPS / tickets / transcripts)
        ↓
  Theme extraction
        ↓
  Problem statement
        ↓
  PRD generation  ←→  Revision loop + AI memory
        ↓
  Dev ticket breakdown
        ↓
  Export / handoff
```
