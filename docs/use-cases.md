# ProductMind Use Cases

## Use Case 1: Post-Discovery PRD Sprint
After 8 customer interviews, PM uploads all notes → ProductMind extracts themes → generates problem statement → produces full PRD → breaks into 14 dev tickets. Time: 9 minutes.

## Use Case 2: NPS Deep-Dive
PM exports 200 NPS responses as CSV → ProductMind clusters verbatims → surfaces top 5 pain themes → generates one problem statement per theme with supporting quotes.

## Use Case 3: Support Ticket Synthesis
Support team exports 3 months of tickets → ProductMind identifies product gaps → maps to existing roadmap items or flags new ones → PM has evidence-backed backlog input in minutes.

## Use Case 4: Sales Call to Requirements
AE shares 6 call transcripts → ProductMind extracts feature requests and pain points → generates a requirements brief the PM can take directly into sprint planning.
